# toolgenlevel-plugin

Public distribution of the **tile-puzzle** Claude Code plugin — a Triple Match
(Tile Explorer / GD_Test) level-design toolkit.

## Install (CLI · VS Code/JetBrains extension · Desktop)
```
/plugin marketplace add thales1020/toolgenlevel-plugin
/plugin install tile-puzzle@toolgenlevel
```
Then use `/toolgenlevel:gen-layout` and `/toolgenlevel:tile-level-design`, or let
Claude invoke them automatically. Requires Python on PATH (the engine is Python).

Full docs: [tile-puzzle/README.md](tile-puzzle/README.md).
